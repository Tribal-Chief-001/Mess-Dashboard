# 🍽️ Mess Intel — Weekly Survival Guide

A sleek, dark-themed dashboard that decodes your hostel mess menu. Know what's being served, rate each dish vibes-based, and figure out if it's worth going to the mess today — or if you should just hit Dakshin.

## ✨ Features

- **Live meal status** — shows which meal is currently being served, or a countdown to the next one
- **7-day full menu** — Breakfast, Lunch, High Tea, Dinner for Mon–Sun
- **Tap-to-rate** — click any dish to cycle through ratings (1–5), saved to localStorage
- **Verdict system** — each day gets a verdict from 🚨 ABORT MISSION to 🏆 LEGENDARY
- **Weekly debrief** — stats like best/worst day, GOAT dish, weekly average
- **Premium dark UI** — Space Grotesk typography, glassmorphism cards, micro-animations

## 🛠️ Tech Stack

- **HTML / CSS / Vanilla JS** — zero dependencies, zero build step
- **Google Fonts** — Space Grotesk, Inter, JetBrains Mono
- **Vercel** — one-click deployment

## 🚀 Run Locally

Just open `index.html` in a browser. That's it. No npm, no webpack, no existential crisis.

Or serve it:
```bash
npx serve .
```

## 📦 Deploy

Already configured for Vercel via `vercel.json`. Push to GitHub and import on [vercel.com](https://vercel.com).
